== Description ==

This plugin enables linkbird customers to transfer their created content directly to their wordpress blog.

Whenever a wordpress post went online, the plugin will inform the linkbird customer instance and update the needed content information like URL or published date.

From that time on linkbird will track the content performance.

== Installation ==

1. Upload all the files to the /wp-content/plugins/ directory or use the Wordpress Admin Panel to install the plugin directly out of the plugin store
2. Activate the plugin in 'Plugins' menu in WordPress
3. Enter your personal installation token on the settings page of the plugin which you will get in your linkbird project wizard
